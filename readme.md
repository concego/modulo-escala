# Módulo Escala 🎵 (Versão PWA)

O **Módulo Escala** é uma ferramenta interativa, inclusiva e profissional projetada para músicos, estudantes e professores. Ela automatiza o cálculo de escalas musicais e campos harmônicos com foco total em **acessibilidade e autonomia**.

Esta nova versão foi transformada em um **PWA (Progressive Web App)**, permitindo que você a instale no seu celular ou computador e a utilize totalmente **offline**.

---

## 🚀 O que há de novo?

*   **Enarmonia Inteligente:** O sistema agora diferencia automaticamente entre Sustenidos (#) e Bemóis (b) com base na tônica escolhida, garantindo precisão teórica.
*   **Catálogo Expandido:** Suporte para escalas Maior, Menor Natural, Menor Harmônica, Menor Melódica, Pentatônicas e Cromática.
*   **Interface Premium:** Novo design de alto contraste com paleta moderna (Slate & Sky), otimizado para leitura visual e navegação por voz.
*   **Acessibilidade Nativa:** Descrições textuais detalhadas em português para leitores de tela (NVDA, TalkBack, VoiceOver).
*   **Instalação PWA:** Funciona como um aplicativo nativo no seu dispositivo.
